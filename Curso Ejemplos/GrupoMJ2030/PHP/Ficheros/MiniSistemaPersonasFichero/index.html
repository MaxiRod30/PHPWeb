Descripción

Comenzamos a desarrollar un sistema que permitirár almacenar datos y generar un nuevo documento con un listado de los mismos. En este caso usamos como medio de almacenamiento un archivo de texto plano generado y gestionado por diferentes procesos en PHP. Como iremos viendo no es la mejor forma de guardar datos, pero nos permitirá entender mejor el uso de los motores de Base de Datos, a la vez que nos prepara para usar los archivos en otras situaciones que no se requiera el uso de BD.
Además comenzamos a incluir todos los conocimientos que hemos adquiridos hasta el momento, esto es, la maquetación de la estructura de los documentos usando XHTML, los estilos con CSS, javascript para controlar formularios y otras funcionalidades, PHP para los proceso que generarán los archivos de datos y documentos como resultados de su lectura.
Esta versión no contempla separación de datos por lo que presentará problemas a la hora de la presentación.
