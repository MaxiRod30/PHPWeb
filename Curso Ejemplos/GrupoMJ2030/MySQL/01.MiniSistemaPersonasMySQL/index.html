Descripción

Comenzamos a desarrollar un sistema que permitirár almacenar datos y generar un nuevo documento con un listado de los mismos. Esta vez utilzamos como medio de almacenamiento la Base de Datos MySQL. Además comenzamos a incluir todos los conocimientos que hemos adquiridos hasta el momento, esto es, la maquetación de la estructura de los documentos usando XHTML, los estilos con CSS, javascript para controlar formularios y otras funcionalidades, PHP para los proceso que generarán los archivos de datos y documentos como resultados de su lectura.
Esta versión no contempla separación de datos por lo que presentará problemas a la hora de la presentación.
