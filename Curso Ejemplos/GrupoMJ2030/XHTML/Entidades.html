Entidades de HTML

Una entidad es un código especial en HTML que permite mostrar símbolos especiales como caracters acentuados y otros.

Todas las entidades comienzan con el símbolo ampersand (&), seguido del código de la entidad, y finaliza con el símbolo punto y coma (;)

Algunos ejemplos

El docente se llama Fernando Núñez

Escapar símbolos

Cuando es necesario mostrar un símbolo que tiene un signficado especial en el leguaje o programa que lo interpreta, será necesario generar un código capaz de mostrar el símbolo pero que no sea interpretado. Esto se conoce como "escapar" un caracter, puesto que se "escapa" del contexto de interpretación.

Para indicar el inicio de párrafo se usa la etiqueta p>, y para indicar el final del mismo la etiqueta </p>
